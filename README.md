# AWS 3-Tier Web Application

## Project Overview

This project demonstrates the deployment of a scalable 3-Tier Web Application on AWS using industry-standard cloud and DevOps practices.

The application is divided into three layers:

- Presentation Tier (React + Nginx)
- Application Tier (Node.js)
- Database Tier (Amazon RDS MySQL)

---

## Architecture

Internet
↓
Nginx (EC2)
↓
Node.js Backend
↓
Amazon RDS (MySQL)
